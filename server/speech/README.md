#### 1. 参数解析 -- 基于gflags模块
 
   1.1 rpc所需信息:

    当前服务器的地址端口：用于搭建rpc服务器的监听地址信息

   1.2 服务注册所需信息:

    注册中心的地址端口：用于进行向服务注册中心进行服务注册

   1.3 语音识别平台所需信息:

    app_id; api_key; secret_key

   1.4 日志模块所需信息:

    运行模式; 日志文件名称; 日志输出级别
